Musicflew

Fullstack application based on React and Node.js that plays some of your uploaded music. You can create artists, albums and and store them in Firebase storage bucket. App has been dockerized. * If you are interested in checking the application please ask me for permission and you will receive a GitLab token from me to get the access. Please note that app is still developing. Thanks !

Technologies Used

ReactJS Tailwind CSS Node.js MongoDB Google Firebase Docker
